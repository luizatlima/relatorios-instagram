Política de Privacidade

Esta Política de Privacidade descreve como o Instagram Analyzer Pro coleta, utiliza e protege as informações obtidas por meio do uso do aplicativo e da integração com a plataforma Meta (Facebook/Instagram).

Este aplicativo é uma ferramenta de análise profissional para gestores e analistas de redes sociais. Não compartilhamos dados com terceiros e não armazenamos informações pessoais de usuários finais do Instagram.

1. Quem Somos

O Instagram Analyzer Pro é um software desktop desenvolvido para auxiliar analistas de redes sociais no monitoramento e análise de perfis do Instagram. O aplicativo utiliza a Instagram Graph API da Meta Platforms, Inc., mediante autorização expressa do titular da conta analisada.

2. Quais Dados Coletamos

Por meio da integração com a Instagram Graph API oficial, o aplicativo pode acessar os seguintes dados da conta Business/Creator autorizada:

  • Nome de usuário e nome do perfil
  • Número de seguidores e contas seguidas
  • Publicações, reels e stories (lista e métricas)
  • Curtidas, comentários e engajamento por publicação
  • Impressões, alcance e visualizações de perfil
  • Cliques no link da bio
  • Crescimento de seguidores ao longo do tempo
  • Menções ao perfil em outras publicações
  • Usuários que comentaram em publicações da conta

Dados que NÃO coletamos:

  • Mensagens diretas (DMs)
  • Histórico de navegação do usuário
  • Dados financeiros ou de pagamento
  • Informações de menores de 18 anos
  • Dados de usuários que não autorizaram o acesso

3. Como Usamos os Dados

Os dados coletados são utilizados exclusivamente para:

  • Gerar relatórios de desempenho do perfil autorizado
  • Calcular métricas de engajamento e crescimento
  • Identificar os melhores horários, formatos e hashtags para publicação
  • Produzir recomendações estratégicas de conteúdo
  • Comparar a evolução do perfil ao longo do tempo

Os dados são armazenados localmente no dispositivo do analista, em banco de dados SQLite protegido. Não há transmissão de dados para servidores externos ou terceiros.

4. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos os dados coletados com terceiros, exceto nas seguintes situações:

  • Com o próprio titular da conta: relatórios em PDF, Excel ou página web podem ser enviados ao cliente titular da conta analisada.
  • Por exigência legal: caso haja determinação judicial ou regulatória aplicável.

5. Segurança dos Dados

Adotamos as seguintes medidas para proteger as informações:

  • Armazenamento local no dispositivo do analista (sem nuvem externa)
  • Access Tokens da Graph API armazenados apenas no arquivo de configuração local
  • Nenhuma credencial é transmitida para servidores de terceiros
  • Acesso restrito ao profissional analista responsável pela conta

6. Retenção de Dados

Os dados são armazenados enquanto o analista mantiver o histórico no aplicativo. O usuário pode excluir qualquer análise ou cliente a qualquer momento diretamente no software. A exclusão remove permanentemente os dados do banco local.

7. Direitos do Titular

O titular da conta Instagram analisada tem direito a:

  • Acesso: solicitar quais dados foram coletados sobre sua conta
  • Retificação: corrigir dados incorretos
  • Exclusão: solicitar a remoção de todos os dados coletados
  • Revogação: revogar o acesso do aplicativo a qualquer momento em Instagram > Configurações > Aplicativos e Sites
  • Portabilidade: receber os dados em formato exportável (JSON/Excel)

8. Uso da Meta Platform (Instagram Graph API)

Este aplicativo utiliza a Instagram Graph API da Meta Platforms, Inc., em conformidade com os Termos de Serviço da Plataforma Meta e a Política de Privacidade da Meta.

O acesso aos dados ocorre somente mediante token de autorização concedido pelo titular da conta Business ou Creator, com as permissões mínimas necessárias: instagram_basic e instagram_manage_insights.

9. Cookies e Rastreamento

O aplicativo desktop não utiliza cookies. A função de coleta de dados via cookies de sessão do Instagram (modo alternativo) utiliza apenas o arquivo cookies.txt exportado pelo próprio usuário de seu navegador, sem qualquer rastreamento de terceiros.

10. Menores de Idade

Este aplicativo é destinado exclusivamente a profissionais de marketing e analistas de redes sociais. Não coletamos intencionalmente dados de menores de 13 anos. Caso identificado, os dados serão excluídos imediatamente.

11. Alterações nesta Política

Esta política pode ser atualizada periodicamente. A data da última atualização será sempre indicada neste documento. O uso continuado do aplicativo após alterações implica aceitação da política atualizada.
